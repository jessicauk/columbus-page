Piazzolla Variable Font
=======================

This download contains Piazzolla as both variable fonts and static fonts.

Piazzolla is a variable font with these axes:
  opsz
  wght

This means all the styles are contained in these files:
  Piazzolla/Piazzolla-VariableFont_opsz,wght.ttf
  Piazzolla/Piazzolla-Italic-VariableFont_opsz,wght.ttf

Get started
-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles
